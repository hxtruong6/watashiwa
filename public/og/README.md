# Open Graph Images

This directory should contain optimized Open Graph images for key pages.

## Recommended Images

- **default.png** (1200x630px) - Default OG image for the site
- **course.png** (1200x630px) - Default OG image for course pages
- **deck.png** (1200x630px) - Default OG image for deck pages

## Specifications

- **Dimensions:** 1200x630px (recommended by Facebook/Open Graph)
- **Format:** PNG or JPG
- **File size:** Keep under 1MB for fast loading
- **Content:** Include WatashiWa branding and relevant imagery

## Dynamic OG Images

For dynamic OG image generation, consider using Next.js ImageResponse API or a service like Vercel OG Image Generation.
