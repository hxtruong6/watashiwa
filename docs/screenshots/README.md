# WatashiWa — Screenshot Gallery

A visual tour of the app. Screens are captured at 1440px desktop width against a
seeded demo account using a Playwright capture script.

## Core experience

### Landing page
![Landing page](01-home.png)
The marketing entry point — "Learn chill, remember long" — introducing the CUBE learning philosophy.

### Dashboard
![Dashboard](02-dashboard.png)
The daily home: study streak, daily goal progress, due-card count, a weekly activity chart, and per-deck progress at a glance.

### SRS study session
![Study session](03-study-session.png)
The spaced-repetition review loop — a focused, distraction-free flashcard with reading and meaning.

### Listen & Type video practice
![Video practice](04-video-practice.png)
Type what you hear from real Japanese video. Fill-in-the-blank and full-sentence modes, kana/kanji answer checking, adjustable playback speed, and per-character hints.

### Knowledge graph
![Knowledge graph](06-knowledge-graph.png)
Vocabulary visualised as a graph of shared-kanji relationships — words that share a kanji are linked, revealing the structure behind the language.

### Kana reference
![Kana reference](07-kana-reference.png)
A clean Hiragana/Katakana reference with romaji, tap-to-play audio, and example words.
