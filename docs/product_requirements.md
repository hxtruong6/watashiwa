# Product Requirements Document (PRD)

## 1. Project Overview

**Project Name:** Watashi JP (JLPT Mastery with SRS)
**Goal:** Build a high-performance vocabulary learning web application that leverages "Golden Time" SRS (Spaced Repetition System) to maximize retention for Japanese learners.
**Target Audience:** Japanese language learners, with a specific focus on those who can benefit from Sino-Vietnamese (Hán Việt) associations (e.g., Vietnamese speakers).

## 2. Core Value Proposition

- **"Golden Time" Optimization:** Unlike standard flashcards, this app calculates the exact moment a user is about to forget a word and schedules a review then.
- **Deep Etymology:** Leverages Hán Việt (Sino-Vietnamese) for stronger memory anchors for Kanji.
- **Active Recall:** Uses interactive exercises (typing, speaking, context) rather than passive card flipping.

## 3. User Stories

- **As a learner**, I want to see a dashboard of reviews due "now" so I can efficiently spend my study time.
- **As a learner**, I want to see the Hán Việt breakdown of a word to help me guess the meaning and remember the Kanji.
- **As a learner**, I want to be tested on writing/typing the word, not just recognizing it, to differentiate between similar-looking Kanji.
- **As a learner**, I want the app to check my sentence construction using AI so I can practice using words in context without a human tutor.

## 4. Success Metrics

- **Retention Rate:** Users should correctly recall >90% of "Mature" cards.
- **Engagement:** Daily Active Users (DAU) competing their review queue.
